# 👟 VOID — Loja Virtual de Tênis Streetwear

## 📖 Sobre o Projeto

A VOID será uma loja virtual especializada em tênis streetwear e sneakers, inspirada no estilo urbano, moderno e minimalista das grandes marcas de skate e moda urbana.

---

# 🏗️ Estrutura Geral do Site

## 🏠 Página Inicial (Home)

A página inicial será responsável por apresentar os principais produtos e campanhas da marca.

## 🖼️ Banner Principal

O site deverá possuir banners grandes e chamativos contendo:

- Lançamentos;
- Novas coleções;
- Promoções;
- Destaques da marca;
- Divulgação de produtos.

Os banners devem utilizar:

- Imagens em alta qualidade;
- Fundo urbano;
- Estilo streetwear;
- Efeitos modernos.

---

# 🧭 Menu de Navegação

O menu ficará localizado na parte superior do site.

## 📂 Itens do menu
- Início;
- Tênis;
- Coleções;
- Promoções;
- Masculino;
- Feminino;
- Contato;
- Minha Conta;
- Carrinho.

Também será necessário:

- Área de login;
- Cadastro de usuários.

---

# 🔎 Sistema de Pesquisa

O site possuirá uma barra de pesquisa para facilitar a busca de produtos.
A barra de pesquisa ficará visível no topo do site.

## 🔍 A pesquisa permitirá:
- Buscar por nome;
- Buscar por categoria;
- Buscar por tamanho.

---

# 📦 Organização dos Produtos

## 🛍️ Catálogo de Produtos

Os produtos serão organizados em formato de grade (grid layout).
Cada produto deverá conter:

- Imagem;
- Nome;
- Preço;
- Botão "Comprar".

---

# 📁 Categorias do Site

O catálogo será dividido em:

- Lançamentos;
- Mais vendidos;
- Promoções;
- Streetwear.

Os produtos serão organizados de forma limpa e moderna.

---

# 👟 Página do Produto

- Fotos em alta qualidade;
- Diferentes ângulos do produto;
- Zoom nas imagens.

### 📄 Informações do produto

- Nome do tênis;
- Preço;
- Descrição;
- Materiais utilizados;
- Tamanhos disponíveis.

### 📏 Seleção de Tamanho

O usuário poderá selecionar:
- Número do tênis;
- Cor do produto.

### 🛒 Botões de Compra

- Adicionar ao carrinho;
- Comprar agora.

---

# 🛍️ Carrinho de Compras

O carrinho permitirá:
- Adicionar produtos;
- Alterar quantidade;
- Remover itens;
- Finalizar compra.

---

# 💳 Sistema de Pagamento

- Cartão de crédito;
- PIX;
- Parcelamento.

---

# 👤 Área do Usuário

O site deverá possuir uma área exclusiva para clientes contendo:

- Histórico de pedidos;
- Dados pessoais.

---

# 📱 Responsividade

O site deverá funcionar corretamente em:

- Computadores;
- Tablets;
- Smartphones.

# 📌 Rodapé (Footer)

O rodapé deverá conter:

A identidade visual da VOID deverá transmitir:
