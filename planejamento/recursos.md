# 🛠️ Recursos do Site VOID

## 📖 Sobre os Recursos do Site

O site da **VOID** contará com diversos recursos para oferecer uma experiência moderna, organizada e intuitiva para os usuários.

Os recursos serão desenvolvidos para facilitar:
- Navegação;
- Pesquisa de produtos;
- Compras online;
- Organização dos pedidos;
- Experiência mobile.

O objetivo será criar uma plataforma prática, rápida e moderna para o público jovem.

---

# 🛒 Carrinho de Compras

O site possuirá um carrinho de compras funcional e organizado.

## 📦 Funções do carrinho

- Adicionar produtos;
- Remover produtos;
- Alterar quantidade;
- Visualizar subtotal;
- Aplicar cupons;
- Finalizar pedidos.

O carrinho também mostrará:
- Imagem do produto;
- Tamanho selecionado;
- Cor escolhida;
- Valor total da compra.

---

# 🔎 Sistema de Pesquisa

O site contará com uma barra de pesquisa para facilitar a busca de produtos.

## 🔍 A pesquisa permitirá

- Buscar por nome;
- Buscar por categoria;
- Buscar por coleção.

O sistema ajudará o usuário a encontrar produtos de forma rápida e prática.

---

# 👤 Área do Usuário

A VOID possuirá uma área exclusiva para clientes.

## 📋 Recursos da conta

- Cadastro de usuários;
- Login;
- Histórico de pedidos;
- Dados pessoais;

A área do usuário permitirá acompanhar informações da conta de maneira organizada.

---

# 💳 Sistema de Pagamento

O site contará com um sistema de pagamento moderno e seguro.

## 💰 Formas de pagamento

- PIX;
- Cartão de crédito;
- Parcelamento.

O checkout será simples e rápido para melhorar a experiência do usuário.

---

# 📱 Layout Responsivo

O site será totalmente adaptado para:
- Celulares;
- Tablets;
- Computadores.
- Navegação simplificada;
- Layout adaptável;
- Botões otimizados;

---

# 🖼️ Sistema de Produtos

Cada produto possuirá uma página própria contendo:

- Imagens em alta qualidade;
- Descrição do produto;
- Preço;
- Tamanhos disponíveis;
- Botões de compra.

---

# 📂 Organização por Categorias

Os produtos serão organizados em categorias para melhorar a navegação.

## 🛍️ Categorias do site

- Lançamentos;
- Mais vendidos;
- Promoções;
- Streetwear.

---

# 📢 Sistema de Promoções

O site contará com um sistema de promoções para divulgação de campanhas.

## 🎯 Recursos promocionais

- Banners promocionais;
- Destaque de lançamentos;
- Divulgação de coleções.

---

# 🌐 Redes Sociais Integradas

O site possuirá integração com as redes sociais da marca.

- Instagram;
- TikTok;
- YouTube.

As redes ajudarão na divulgação da marca e dos produtos.

---

# 🚀 Conclusão

Os recursos da VOID serão desenvolvidos para oferecer uma experiência moderna, prática e organizada para o público jovem.

O site contará com:
- Sistema de compras;
- Pesquisa rápida;
- Layout responsivo;
- Segurança;
- Navegação intuitiva;
- Organização visual moderna.

A proposta será criar uma plataforma eficiente e conectada à cultura streetwear e digital.
