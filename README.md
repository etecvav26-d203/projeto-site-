# 🌐 Resumo da Marca e do Site VOID

## 🎯 Objetivo

A **VOID** será uma marca de tênis streetwear criada para unir moda urbana, modernidade e identidade jovem.

O principal objetivo da marca será oferecer:
- Produtos estilosos;
- Conforto;
- Experiência moderna;
- Forte identidade visual;
- Conexão com o público jovem.

---

# 🏗️ Estrutura do Site

O site da VOID será organizado de forma moderna e intuitiva.

A plataforma contará com:
- Página inicial;
- Catálogo de produtos;
- Sistema de pesquisa;
- Carrinho de compras;
- Área do usuário;
- Sistema de pagamento;
- Layout responsivo.

O objetivo será facilitar a navegação e melhorar a experiência dos usuários.

---

# 👥 Faixa Etária

A VOID será voltada principalmente para:
- Pré-adolescentes;
- Adolescentes;
- Jovens ligados à cultura urbana.

O público da marca buscará:
- Estilo streetwear;
- Moda moderna;
- Sneakers;
- Visual urbano;
- Produtos confortáveis.

---

# 🎨 Identidade Visual

A identidade visual da VOID será baseada em:
- Minimalismo;
- Modernidade;
- Cultura streetwear;
- Estilo urbano.

## 🖤 Paleta de cores

- ⚪ Branco — cor dominante (60%);
- ⚫ Preto — cor secundária (30%);
- 🟣 Roxo — cor destaque (10%).

A fonte principal do site será a **Poppins**.

---

# 📱 Mobile First

O site será desenvolvido utilizando o conceito **Mobile First**, priorizando celulares e dispositivos móveis.

O layout será:
- Responsivo;
- Rápido;
- Organizado;
- Adaptável para smartphones.

Essa estratégia será importante porque o público da VOID utiliza principalmente celulares para acessar a internet e realizar compras.

---

# 🚀 Proposta da Marca e do Site

A proposta da VOID será criar uma experiência moderna e conectada à cultura urbana.

O site permitirá:
- Visualizar produtos;
- Pesquisar tênis;
- Comprar online;
- Explorar lançamentos;
- Navegar por categorias.

A marca buscará transmitir:
- Personalidade;
- Modernidade;
- Estilo urbano;
- Identidade jovem.
